## **Project: CloudWatch Monitoring Dashboard for ECS Application**

**Summary:**  
Deployed a containerized application (Nginx) on ECS Fargate, configured task CPU and memory settings, and created a CloudWatch dashboard to monitor real-time performance metrics under simulated load.

**Key Skills:**  
- CloudWatch Dashboards  
- ECS Performance Tuning  
- Monitoring & Observability
