## **Project: Serverless Automation with S3 Event Triggers & Lambda**

**Summary:**  
Configured an S3 bucket to automatically trigger a Lambda function when new files are uploaded. Developed the Lambda function to log event details to CloudWatch Logs. Verified successful execution through CloudWatch log entries and SES email alerts.

**Key Skills:**  
- AWS Lambda  
- S3 Event Notifications  
- CloudWatch Logs  
- Serverless Computing  
- SES Email Alerts
