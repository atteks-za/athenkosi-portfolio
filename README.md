

# **AWS Bootcamp Projects Portfolio**

This branch showcases my **AWS Bootcamp Projects**, covering 10 weeks of hands-on cloud engineering, infrastructure deployment, monitoring, and serverless automation. Each project demonstrates practical skills in AWS services, DevOps practices, and cloud security.

---

## **Week 1 – AWS Account Initialization & Environment Setup**

**Project:** AWS Account Initialization & Environment Setup
**Summary:** Created and configured a new AWS account, activated billing details, and validated console access.
**Key Skills:** AWS Account Management, IAM Basics, Cloud Onboarding

## **Week 2 – Identity & Access Management with AWS Identity Center**

**Project:** IAM with AWS Identity Center
**Summary:** Configured AWS Identity Center, created users, and assigned SecurityAudit permission sets following least-privilege principles.
**Key Skills:** Identity Center, Permission Sets, IAM Governance, Security Best Practices


## **Week 3 – Deploying a Secure Windows EC2 Instance**

**Project:** Secure Windows EC2 Deployment
**Summary:** Launched Windows Server 2019 EC2 instance in a public subnet, restricted RDP access to a single IP, and validated connectivity.
**Key Skills:** EC2, Security Groups, Networking, Remote Desktop


## **Week 4 – Multi-VPC Networking & Peering Configuration**

**Project:** Multi-VPC Networking & Peering
**Summary:** Created two VPCs with unique CIDR ranges, configured public/private subnets, and established VPC peering with updated route tables.
**Key Skills:** VPC Design, Subnets, Routing, VPC Peering, Network Architecture


## **Week 5 – Containerized Grafana Dashboard on ECS Fargate**

**Project:** Grafana on ECS Fargate
**Summary:** Deployed Grafana using Docker on ECS Fargate, exposed port 3000, configured security groups, and accessed the dashboard publicly.
**Key Skills:** ECS Fargate, Docker, Security Groups, Cloud Monitoring


## **Week 6 – Metabase on ECS with RDS PostgreSQL**

**Project:** ECS-Hosted Metabase with PostgreSQL RDS
**Summary:** Launched Metabase on ECS Fargate connected to a PostgreSQL RDS instance in the same VPC using secure SG rules on port 5432.
**Key Skills:** ECS, RDS PostgreSQL, Environment Variables, Secure App-to-DB Communication



## **Week 7 – CloudWatch Monitoring Dashboard for ECS Application**

**Project:** ECS App Monitoring with CloudWatch
**Summary:** Deployed a containerized app (Nginx/Grafana), configured CPU/memory settings, and built a CloudWatch dashboard tracking real-time metrics.
**Key Skills:** CloudWatch Dashboards, ECS Performance Tuning, Monitoring & Observability


## **Week 8 – Static Portfolio Website with S3 & CloudFront**

**Project:** Static Website Hosting
**Summary:** Hosted a static portfolio website on S3, applied bucket policies, and delivered it securely via CloudFront CDN.
**Key Skills:** S3 Hosting, CloudFront CDN, Bucket Policies, Front-End Deployment


## **Week 9 – Secure Custom Domain Hosting with Route 53 & ACM**

**Project:** Custom Domain + HTTPS Deployment
**Summary:** Registered a custom domain, configured Route 53, requested SSL certificates via ACM, and served content securely over HTTPS with CloudFront.
**Key Skills:** Route 53 DNS, CloudFront CDN, ACM SSL, HTTPS Configuration, S3 Hosting


## **Week 10 – Serverless Automation with S3 Event Triggers & Lambda**

**Project:** Serverless Automation (S3 → Lambda → SES/CloudWatch)
**Summary:** Configured an S3 bucket to trigger a Lambda function for logging events or sending SES notifications. Verified execution via CloudWatch Logs or SES.
**Key Skills:** AWS Lambda, S3 Event Notifications, CloudWatch Logs, Serverless Computing, SES
