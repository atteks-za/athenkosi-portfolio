## **Project: Multi-VPC Networking & Peering Configuration**

**Summary:**  
Built two VPCs with unique CIDR ranges, created public and private subnets in each VPC, and established VPC peering. Updated route tables to enable secure communication between the two networks.

**Key Skills:**  
- VPC Design  
- Subnet Configuration  
- Routing  
- VPC Peering  
- Network Architecture
