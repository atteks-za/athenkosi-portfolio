## **Project: Static Portfolio Website with S3 & CloudFront**

**Summary:**  
Hosted a static portfolio website on Amazon S3, enabled public access using secure bucket policies, and deployed a CloudFront distribution to deliver the website globally with improved performance and HTTPS security.

**Key Skills:**  
- S3 Website Hosting  
- CloudFront CDN  
- Bucket Policies  
- Front-End Deployment
